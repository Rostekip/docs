---
title: "Edit a product"
description: "Once a product is available, you can always edit it later."
---

To edit a product, go to the "Products" item in the header and press on "Edit" in the list of projects. Be sure to select a project first on top.

<img
  src="/images/product3.png"
  alt="addproject"
  height="100"
  className="rounded-lg"
/>

A popup will appear where you will be able to make the necessary changes

<img
  src="/images/product8.png"
  alt="addproject"
  height="100"
  className="rounded-lg"
/>

Pressing on the Update button will allow you to save the changes.
