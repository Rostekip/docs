- Edit: See Edit a product
- Prices: Pressing on this button will go to the "Prices" section and will only show the prices for this product.
- Translations: Each product has a default translation. But if you wish to add other translations, you can do so be clicking on this button.
- Test Payment: Only when the prices are filled in, you will be able to test the payment page for this product.
