---
title: "Edit a price"
description: "Prices can always be edited."
---

In order to edit a price, select the "Prices" on top. You will need to select a project on top in order to create a price.

Click on 'Edit' in the list to edit a price for a product

<img
  src="/images/editprices2.png"
  alt="addproject"
  height="100"
  className="rounded-lg"
/>

The popup will appear where you will be able to add, edit or remove the price of product(s)
