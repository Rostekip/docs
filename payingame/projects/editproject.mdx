---
title: "Edit a project"
description: "Once a project is available, you can always edit it later."
---

To edit a project, go to the "Projects" item in the header and press on "Edit" in the list of projects.

<img
  src="/images/project3.png"
  alt="addproject"
  height="100"
  className="rounded-lg"
/>

A popup will appear where you will be able to make the necessary changes

<img
  src="/images/project4.png"
  alt="addproject"
  height="100"
  className="rounded-lg"
/>

Pressing on the Update button will allow you to save the changes.
